# 面试题 16.26. 计算器
```
给定一个包含正整数、加(+)、减(-)、乘(*)、除(/)的算数表达式(括号除外)，计算其结果。
表达式仅包含非负整数，+， - ，*，/ 四种运算符和空格  。 整数除法仅保留整数部分。
```

## 示例 1:
```
输入: "3+2*2"
输出: 7
```
## 示例 2:
```
输入: " 3/2 "
输出: 1
```
## 示例 3:
```
输入: " 3+5 / 2 "
输出: 5
```
## 说明：
```
你可以假设所给定的表达式都是有效的。
请不要使用内置的库函数 eval。
```

```c++
class Solution {
public:
    int calculate(string s) {
        std::stack<int> num;
        std::stack<char> op;
        int num_tmp = 0;
        for(int i = 0;i < s.size();++i)
        {
            if(s[i] >= '0' && s[i] <= '9')
            {
                while(s[i] >= '0' && s[i] <= '9')
                {
                    num_tmp = num_tmp * 10 + (s[i] - '0');
                    i++;
                }
                i--;
                num.push(num_tmp);
                num_tmp = 0;
            }
            else if(s[i] != ' ')
            {
                while(!op.empty() && priority(s[i]) <= priority(op.top()))
                {
                    int n2 = num.top();
                    num.pop();
                    int n1 = num.top();
                    num.pop();
                    char tmp = op.top();
                    op.pop();
                    num.push(operation(n1,n2,tmp));
                }
                op.push(s[i]);
            }
        }
        while(!op.empty())
        {
            int n2 = num.top();
            num.pop();
            int n1 = num.top();
            num.pop();
            char tmp = op.top();
            op.pop();
            num.push(operation(n1,n2,tmp));
        }
        return num.top();
    }
    int priority(char c)
    {
        switch(c)
        {
            case '+':
            case '-':
                return 1;
            case '*':
            case '/':
                return 2;
            default:
                return 0;
        }
    }
    int operation(int n1,int n2,char op)
    {
        switch(op)
        {
            case '+':
                return n1+n2;
            case '-':
                return n1-n2;
            case '*':
                return n1*n2;
            case '/':
                return n1/n2;
            default:
                return 0;
        }
    }
};
```
